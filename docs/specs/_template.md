## 頁面說明

### 列表頁（`/feature`）

**功能需求：**
- [ ] 顯示資料表格，欄位：名稱、狀態、建立時間、操作
- [ ] 頂部搜尋欄，依名稱篩選
- [ ] 右上角「新增」按鈕，點擊開啟 Modal
- [ ] 每頁顯示 10 筆，底部分頁

**操作行為：**
- 點擊「編輯」→ 開啟編輯 Modal
- 點擊「刪除」→ 顯示確認提示，確認後從列表移除（假資料操作）

---

### 新增/編輯 Modal

**表單欄位：**
| 欄位 | 類型 | 必填 | 驗證規則 |
|------|------|------|---------|
| 名稱 | text input | 是 | 最少 2 字 |
| 狀態 | select | 是 | active / inactive |
