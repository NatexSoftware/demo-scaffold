## 功能說明

管理系統內部使用者帳號，支援查看、新增、編輯、停用等操作。此為後台管理功能，僅限管理員使用。

## 頁面說明

### 使用者列表（`/users`）

**功能需求：**
- [ ] 表格欄位：頭像+姓名、電子郵件、角色、狀態（Badge 顯示）、建立時間、操作
- [ ] 頂部搜尋欄，依姓名或 email 篩選
- [ ] 狀態 Badge：active = 綠色「啟用」、inactive = 灰色「停用」
- [ ] 角色 Badge：admin = 藍色、editor = 紫色、viewer = 灰色
- [ ] 每頁 10 筆，底部分頁

**操作行為：**
- 點擊「編輯」→ 開啟編輯 Modal（僅 UI，不實際儲存）
